# Docker React Node App

## Overview
This project demonstrates how to containerize a full-stack React and Node.js application using Docker. It was created as part of my DevOps training to practice building, running, and managing containerized applications.

## Technologies
- Docker
- React.js
- Node.js
- Linux
- Git

## Features
- Containerized React frontend
- Containerized Node.js backend
- Docker image creation
- Docker container deployment
- Local development using Docker

